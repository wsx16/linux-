# 智能售货机系统项目简历

## 项目名称：智能售货机系统（25061VendMachine）

## 技术栈
- **开发框架**：Qt 5.x + C++
- **网络通信**：Qt Network模块（QNetworkAccessManager、QNetworkRequest、QNetworkReply）
- **加密安全**：OpenSSL库（RSA加密、SHA哈希、Base64编解码）
- **UI设计**：Qt Widgets
- **第三方集成**：支付宝开放平台API、二维码生成库（qrcodegen）

## 工程实现
本项目采用模块化架构设计，主要分为UI界面模块和支付宝支付模块两大核心部分。

**UI界面模块**：
- 实现商品展示（Shopping）和支付（Pay）两个主要页面
- 基于Qt Widgets构建用户友好的图形界面
- 支持页面间的无缝切换与交互

**支付宝支付模块**：
- 封装支付宝开放平台API，实现订单创建与查询功能
- 使用RSA非对称加密算法对交易数据进行签名和验证
- 通过QNetworkAccessManager与支付宝服务器进行HTTPS通信
- 实现二维码动态生成与显示功能
- 采用定时器轮询机制实时监控支付状态

**核心功能流程**：
1. 用户在商品页面选择商品
2. 系统调用支付宝接口创建支付订单
3. 生成并显示支付二维码
4. 实时查询支付状态
5. 支付成功后触发相应的商品出货流程

该系统成功实现了智能售货机的核心功能，具备良好的安全性、稳定性和用户体验。
